将githubpage和域名绑定
2024-04-21

0.关于哥们遇到的两个问题

Both lotus.hi.cn and its alternate name are improperly configured Domain does not resolve to the GitHub Pages server. For more information, see documentation (NotServedByPagesError).

www.lotus.hi.cn is improperly configured Your site’s DNS settings are using a custom subdomain, www.lotus.hi.cn, that is not set up with a correct CNAME record. We recommend you change this to a CNAME record pointing to lijinfengisrunning.github.io. For more information, see documentation (InvalidCNAMEError).

解决如下

1. 首先ping一下

ping example.github.io

确定是ipv4还是ipv6的地址

2. 然后建一下

找到githubpage的文档中官方给的相应ip地址

将对应的IP地址添加到域名提供商的域名解析中

然后添加CNAME,主机记录填写www,记录名填写example.github.io

3. 最后建一下

在blog的source目录下,新建一个CNAME

哥们用的是hexo,建一个这个文件是为了防止每次在部署时,把域名删掉

分钟左右,再check again,就瑟克赛斯否了
